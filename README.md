# Daupso

Simple e-commerce website

## Features

- User login
- Product search
- Admin control
  - User management
  - Product management

## Installation

1. Clone the repository:

```bash
git clone https://https://github.com/young170/Daupso.git
```

2. Navigate to the project directory:

```bash
cd daupso
```

3. Install dependencies:

```bash
npm install
```

## Usage

Run the backend application:

```bash
cd daupso_backend
node index.js
```

Run the frontend application:

```bash
cd daupso_frontend
npm run dev
```
